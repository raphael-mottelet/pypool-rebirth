#pypool 2024 ESIEE-IT

Le but de cet exercice est de créer un jeux vidéo, sur le thème des pyramides et de l'égypte, dans le cadre de la piscine python ESIEE-IT.

Utilisez Z,Q,S,D pour vous déplacer dans le jeux.
Utilisez Z,S Touche haut, Touche bas pour vous déplacer dans le menu et entrée pour selectionner une option.
Utilisez echap pour mettre en pause le jeux.
